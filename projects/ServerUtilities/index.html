ServerUtilities

Forty 2022-07-DD

ServerUtilities is a mod I made for Titanfall 2 + Northstar. It implements a both modder and user friendly chat command and poll framework with extra features such as team balance, chat filter and more. The mod is divided into independent modules which can be disabled / enabled providing an easy way to diagnose collisions with other mods and allowing easy modification.
